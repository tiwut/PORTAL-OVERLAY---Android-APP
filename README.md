# Tiwut Portal Overlay

An Android overlay launcher that allows you to access your favorite apps and settings from anywhere on your device via a floating window.

## Features

- **Floating Overlay**: Access the launcher over any app.
- **Quick Settings Tile**: Easily toggle the overlay from your device's Quick Settings.
- **Customizable**: (Add specific customization features here if applicable).
- **Material 3 Design**: Modern and clean UI using Jetpack Compose.

## Getting Started

### Prerequisites

- Android Studio Ladybug or newer.
- Android SDK 24 (Nougat) or higher.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/tiwut/PORTAL-OVERLAY---Android-APP.git
   ```
2. Open the project in Android Studio.
3. Build and run the app on your device or emulator.

### Usage

1. Grant the "Display over other apps" permission when prompted.
2. Open the app to configure your launcher items.
3. Use the Quick Settings Tile to show/hide the overlay.

This project does not collect or transmit any personal data. All settings and launcher data are stored locally on your device.
